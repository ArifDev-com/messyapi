# About
This is a Api as a service project that provide API connection to Whatsapp and Facebook messenger api. This will be a 
automated bot system for taking long orders from customers.

# Architecture
This project is built with Laravel v12 and PHP v8.2.
used composer packages:  
"php": "^8.2",  
"doctrine/dbal": "^4.4",  
"laravel/framework": "^12.0",  
"laravel/tinker": "^2.10.1",  
"netflie/whatsapp-cloud-api": "^3.1"  
For frontend, used Bootstrap v4 and jquery v3.
